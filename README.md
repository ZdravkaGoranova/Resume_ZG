### Installation Steps

1. Clone the repository:

   ```bash
   git clone https://github.com/ZdravkaGoranova/Resume_ZG.git
   cd Resume_ZG
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server.js:

   ```bash
   npx nodemon server.js
   ```
